## Perform below instructions:

- make: build excutable file.
- make clean: clean everything.

## How to run:
### With bai1:
```bash
./bai1/bai1
```
### With bai2:
#### write data to file 
```bash
 ./bai2/bai2 filename numb_bytes w data_want_write
 ```
#### read data to file 
```bash
/bai2/bai2 filename numb_bytes r data_want_write
 ```
### With bai3:
 ```bash
./bai3/bai3
 ```
